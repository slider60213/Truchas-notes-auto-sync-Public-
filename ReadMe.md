# 2. 部分排版錯誤
- 由於 Github 幾年前取消了 Markdown 語法中對部分排版的支援，網頁上看到的文字及圖片其位置、顏色、大小可能無法正確呈現。
- 可以直接下載 md 檔然後用 VS CODE 或是 Obsidian (建議) 開啟觀看。

# 3. 頁面跳轉錯誤
- 頁面跳轉有時候不精確，要根據上下文或是網址末端標註的位置來尋找。
- 例如跳轉後網址是 `https://github.........md#turn-40` ，表示要找第 Turn 40 的問答。
