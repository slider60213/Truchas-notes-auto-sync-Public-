# 🦖 以前

完成 inp 檔設置後：

 1. 輸入模擬指令。
 2. 依照 Output_dt 將不同時間步的結果記錄成 bin 檔。
 3. bin 檔可以透過轉檔指令 轉換成 gmv/tecplot/ensight 檔。
 4. 模擬終止後，若要接續模擬需要先透過轉檔指令生成 restart 檔。
 5. 執行 Restart 前需要先將 Output 資料夾改名避免檔案覆寫。
 6. 輸入 Restart 的指令。
 7. 重複步驟2.、3 來得到新的模擬成果。
 8. 將 gmv/tecplot/ensight 檔移到本地電腦。
 9. 由於這些檔案在 Restart 後也是從 0 開始編號，需要先批量改檔名。
10. 通常會有 4～5個 Output 資料夾的 gmv 檔要重新編號，千萬不要改錯。
11. 完成後才可以用 Matlab 或 Python 讀取。
12. 若模擬途中為了確認成果先轉檔輸出，模擬完成後需要指定輸出的區段或是直接重複輸出。
13. 若是在 Restart 接續模擬途中提前轉檔，還需要再重新編號。
---
# 👨‍💻 以後

完成 inp 檔設置後：

1. 輸入 `./MakeProblemAuto.sh`。
2. 依照錯誤訊息修正指令為 `./MakeProblemAuto.sh 1 Bingham5.inp`。
3. 直接拉到本地電腦用 Matlab 或 Python 讀取。

<center><font color="#ffff00">不用轉檔、不用改資料夾名稱、不用編號、不用管 Restart。</font></center>

# 📝 內容紀錄


## Ver.4
---------

## 優點：

1. 你唯一需要做的只有跑模擬 。

2. 以下功能自動化：
	- Output 資料夾自動編號，Restart 後亦自動接續編號。
	- 依照當前 bin 檔數量生成對應的 gmv/tecplot 檔，Restart 後亦自動接續編號。
	- 模擬會依照 Output 資料夾編號來決定是初始模擬還是 Restart 接續模擬。
	- 模擬終止（無論正常與否），都會自動生成 restart 檔。
	- 只要重新執行 `MakeProblemAuto.sh` 即可 Restart 接續模擬。

3. 可以調整為輸出其他格式 tecplot、ensight。
---
## 缺點 ：

1. 更改 .sh 檔內容後需要 fix 才會生效，可以全部改寫成 .py 檔來避免。
2. 目前輸出 gmv 檔以外的格式沒有重新編號，可以比照範例來改寫。

---
## 使用說明：

### 如果你是小白
	1.  到帳號底下的.bashrc新增以下兩行，記得全選右鍵改屬性777，存檔後重開terminal讓改動生效
			# 自定義修復 Windows 傳入腳本的快捷指令
			alias fix='f(){ sed -i "s/\r$//" "$1"; chmod +x "$1"; }; f'
	2.  直接到'/media/Lab1/01_Truchas files/Shane' 
	3.  下載 truchas-2.5.3_Auto_Ver4.tar.gz
	4.  輸入 tar zxf truchas-2.5.3_Auto_Ver4.tar.gz 來解壓縮
	5.  解壓縮之後全選右鍵改屬性777
	6.   cd truchas-2.5.3_Auto_Ver4
		~/lamboot.sh
		./MakeClean.sh
		./Parallel.sh 或 ./Serialsh 進行編譯
	7.   cd problem/test
	8.  執行 MakeProblemAuto.sh來跑模擬
		直接執行./MakeProblemAuto.sh，此為錯誤輸入，但可以看顯示的提示來修正指令
		正確用法為./MakeProblemAuto.sh 1 Bingham5.inp

	9.  輸出格式改為tecplot或ensight要到AutoGetResult.sh的第13～15行
		像下方範例把對應的格式取消註解，改完記得fix AutoGetResult.sh

			#OUT_FORMAT="gmv" 
			OUT_FORMAT="tecplot" 
			#OUT_FORMAT="ensight" 

	10.  可以單獨執行AutoGetResult.sh來手動輸出，善用TAB補齊路徑，路徑末尾有無'/'皆可
		直接執行./AutoGetResult.sh，此為錯誤輸入，但可以看顯示的提示來修正指令
		正確用法為:
			輸出特定區間 ./AutoGetResult.sh Bingham5_output/ 0 600
			輸出最後一步 ./AutoGetResult.sh Bingham5_output/ -1
			輸出全部時間 ./AutoGetResult.sh Bingham5_output/ ALL
